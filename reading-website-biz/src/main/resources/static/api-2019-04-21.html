查询反馈信息

基本信息

Method： POST

接口描述：

请求参数

Headers

参数名称 参数值 是否必须 示例 备注

messagestring必须请求信息

├─ pageNuminteger必须当前页号
├─ pageSizeinteger必须分页大小
